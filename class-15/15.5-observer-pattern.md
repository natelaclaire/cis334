# 15.5 Observer Pattern

The **Observer pattern** is about notifications. One object watches another. When something changes, observers are automatically informed.

This is useful for loggers, event systems, notifications, or updating different parts of the UI.

```php
<?php
class Event {
    private array $listeners = [];

    public function attach(callable $listener) {
        $this->listeners[] = $listener;
    }

    public function trigger($data) {
        foreach ($this->listeners as $listener) {
            $listener($data);
        }
    }
}

$event = new Event();
$event->attach(function($data) { echo "Listener 1: $data"; });
$event->attach(function($data) { echo "Listener 2: $data"; });

$event->trigger("User registered.");
```
